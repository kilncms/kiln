# Kiln starter templates — catalog of 20 (build 5 first)

Strategy (locked): starters are FINISHED Kiln-ready sites — pick → live → edit. The AI lane
(Lovable/Claude/v0 restyle) is optional customization, never a required step. Use
open-design/bravebrand as inspiration + component kits, not clones of recognizable brands.

## Build first (the launch five — one per major audience)
1. **Maple** — small business / maker (exists: kiln-demo). Hero, services repeat, journal, members.
2. **Atelier** — artist/photographer portfolio. Full-bleed gallery repeat, statement page, CV, inquiry.
3. **Counter** — restaurant/café. Menu as repeat blocks (sections→items), hours/location, gallery, specials post.
4. **Practice** — consultant/therapist/lawyer. Credibility-first: bio, services, FAQ repeat, intake CTA, resources members area.
5. **Commons** — nonprofit/civic org (Atlanta Commons shape). Mission, programs repeat, events journal, documents members area, donate links.

## The other fifteen (paper designs; promote on demand)
6. Stage — musician/band: shows repeat, embeds, EPK members area
7. Chapel — church/community group: sermons journal, ministries repeat, bulletin members
8. Ledger — bookkeeping/accounting firm: services, pricing table repeat, client portal members
9. Trades — plumber/electrician/landscaper: before/after gallery, service areas, reviews repeat
10. Shears — salon/barber: services+prices repeat, stylists repeat, booking links
11. Mat — yoga/fitness studio: class schedule repeat, instructors, pricing, member resources
12. Roast — coffee/retail product: product grid repeat, story, stockists, wholesale members
13. Inkwell — author/writer: books repeat, essays journal, press kit, newsletter CTA
14. Lens — wedding/event photographer: package pricing, galleries per event, client galleries members
15. Keynote — speaker/coach: talks repeat, testimonials, booking, media page
16. Stoop — real-estate agent: listings repeat, sold gallery, neighborhood guides journal
17. Clinic — dentist/med practice: services, providers repeat, patient forms members (PDFs!)
18. Forge — agency/freelancer studio: case studies repeat+pages, capabilities, careers
19. Almanac — local news/newsletter: journal-first, sections, archive, supporters members
20. Festival — event/conference: schedule repeat, speakers repeat, venue, attendee info members

Each ships with: annotated pages, _templates/ (post, page, card), members area pre-wired,
style palette in kiln-config, README + "restyle me" AI prompt tuned to that template.
Source kits: github.com/nexu-io/open-design, playground.bravebrand.com/sites (inspiration only).
