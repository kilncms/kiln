The fine print, plainly.

Is it really free? What's the catch?

The software is MIT open source. GitHub repos are free. Cloudflare Pages' free tier (unlimited static bandwidth, commercial use allowed) hosts the site, and a free Cloudflare Worker handles sign-in. The "catch" is that you self-host those two free things — the wizard does it in about ten minutes.

Do my editors need GitHub accounts?

No. The owner uses GitHub (it's their repo). Everyone else signs in with Google — you add their email and a role. There are also one-time links for people without Google. Their changes are committed on their behalf, attributed by name.

Can editors break my design?

That's the whole point: editors change content, not layout. Text, images, links, list items, menu entries — yes. Dragging boxes around or changing fonts arbitrarily — deliberately not. Styles come from a palette you define in CSS.

Where does my content live? Can I leave?

In your own GitHub repository, as plain HTML with full version history. "Leaving Kiln" means deleting two script tags. There's no export, because nothing was ever imported.
